
# 子音で終わらせる発音

|     |  a  |  i  |  u  | (u) |  e  |  o  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| (x) |  あ  |  い  |  う  |     |  え  |  お  |
|  k  |  か  |  き  |  く  |     |  け  |  こ  |
|  s  |  さ  |  し  |     |  す  |  せ  |  そ  |
|  t  |  た  |     |     |     |  て  |  と  |

| ts  |     |     |     |  つ  |     |     |

|  w  |  わ  |     |     |     |     |  を  |

上の表の通り、音を出す方法は基本的に子音と母音の組み合わせだ。

- 子音
	- 音を出す前の口の形
- 母音
	- 音を出す仕方

日本語は基本的に子音一つ、母音一つで音を出す。

でも音が終わる時子音を入れることも十分可能だ。

- 子音（はじめ）
	- 音を出す前の口の形
- 母音
	- 音を出す仕方
- 子音（おわり）
	- 音を終わらせる時の口の形

日本人が慣れてない発音はこのタイプの発音である。

終わりの子音を出す理論自体は意外と簡単。

音が終わる時、はじめの子音を真似すればいい。

もちろん練習は必要。

下の表は反対側でも子音を追加した表である。
