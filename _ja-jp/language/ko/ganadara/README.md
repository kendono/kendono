
# ガナダラ

韓国語の一文字は基本的に2種類

- 子音＋母音
	- 例：가 (が)
- 子音＋母音＋子音
	- 例：각 (「がくせい」の「がく」)

ガナダラは母音を「ㅏ(あ)」に固定して、
そのあと色んな子音を使って子音を覚えるコツ

| No. |  ㄱ  |  ㅏ  |  あ  |
| --: | :-: | :-: | :-: |
|   1 |  ㄱ  |  가  |  が  |
|   2 |  ㄴ  |  나  |  な  |
|   3 |  ㄷ  |  다  |  だ  |
|   4 |  ㄹ  |  라  |  ら  |
|   5 |  ㅁ  |  마  |  ま  |
|   6 |  ㅂ  |  바  |  ば  |
|   7 |  ㅅ  |  사  |  さ  |
|   8 |  ㅇ  |  아  |  あ  |
|   9 |  ㅈ  |  자  | じゃ  |
|  10 |  ㅊ  |  차  | ちゃ  |
|  11 |  ㅋ  |  카  |  か  |
|  12 |  ㅌ  |  타  |  た  |
|  13 |  ㅍ  |  파  |  ぱ  |
|  14 |  ㅎ  |  하  |  は  |

形を覚えることが難しいだと思うかも知れないが
韓国語の文字「ハングル」は
発音する時の口や舌の形を参考して作った文字である。

例えば「가(が)」っていうときは
舌が実際に口の上につく。

似てる形同士で集めてみるとこうなる：

| No. |     |     |     |
| --: | :-: | :-: | :-: |
|   1 |  가  |  카  |     |
|   2 |  나  |     |     |
|   3 |  다  |  타  |     |
|   4 |  라  |     |     |
|   5 |  마  |  바  |  파  |
|   6 | (바) |     |     |
|   7 |  사  |  자  |  차  |
|   8 |  아  |  하  |     |
|   9 | (자) |     |     |
|  10 | (차) |     |     |
|  11 | (카) |     |     |
|  12 | (타) |     |     |
|  13 | (파) |     |     |
|  14 | (하) |     |     |

これなら多分もっと簡単に覚えるはず。

読んでみると、形が似ている文字を集めただけなのに、
結構お互いに発音が似てたり
音を出すコツが似ていることを気づくことができると思う。

ここでは子音の「発音」にはあんまり注目しない。
一応子音の「形」を覚えることに集中する。
