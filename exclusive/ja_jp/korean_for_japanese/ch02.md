
# 英語で読んでみるハングル

ハングルには子音と母音っていうやつがあります。基本的に子音は14個、母音は10個があります。

ハングルは「子音+母音+子音」の順で一つの文字を作ります。最後の子音はある可能性あって、ない可能性もあります。

じゃ作れる文字の数は、「14x10x14=1960」なんでしょうか。それは違います。最後の子音はない可能性もあるから、「14x10x(14+1)=2100」に計算する必要があります。それだけじゃなく、これはただ基本的な子音と母音の数を言っただけなので、バリエーションもカウントしたらその数はもっと増えます。なので正確な計算式は、「(14+5)+(10+11)+((14+13)+1)=11172」です。多いですよね？このせいでハングルは文字を表記に使うユニコードっていうシステムの中でも、漢字の次に多い文字としても有名です。もちろん、漢字はユニコードで10万個くらいはあるから、比べるほどではないんですけどね。

それではハングルに具体的にどんな文字があるのか調べてみましょう。言った通りに、英語を使って読み方を覚える予定です。

まず、子音です。韓国人はよく「가나다라(ガナダラ)」と言いながらこれを覚えます。

| 番号  | 子音  | 日本語 | 英語  |
| :-: | :-: | :-: | :-: |
| 01  |  가  |  が  | ga  |
| 02  |  나  |  な  | na  |
| 03  |  다  |  だ  | da  |
| 04  |  라  |  ら  | ra  |
| 05  |  마  |  ま  | ma  |
| 06  |  바  |  ば  | ba  |
| 07  |  사  |  さ  | sa  |
| 08  |  아  |  あ  | ah  |
| 09  |  자  |  ざ  | ja  |
| 10  |  차  | つぁ  | cha |
| 11  |  카  |  か  | ka  |
| 12  |  타  |  た  | ta  |
| 13  |  파  |  ぱ  | pa  |
| 14  |  하  |  は  | ha  |

旅部分はほとんど似てる音を出しています。でも少し説明が必要なのは「자」と「차」です。これは日本語でも英語でも説明が難しいかも知れません。

一応、「ざ」と「za」の発音は本当に似ています。似てすぎて、簡単に同じ発音だと思っても特に問題ないくらいです。でも英語の「ja」は、正確には「じゃ」の発音ではありません。発音するコツが「ざ」と違うから「じゃ」になるんだと思うかも知れないんですけど、「じゃ」に似ている発音は「jya」です。韓国語の場合は「쟈」です。

ならば「자」と「ja」は同じ発音を持っているんでしょうか。それもまたちがいます。韓国語の「ㅈ(ジウッ)」は、「jar」の「j」よりは「gym」の「g」に似ている発音です。軽く音をだす「j」よりは、もっと低くて重い発音っていう意味です。だから韓国語の「자」は「ja」と「za(ざ)」、その間にある発音だと思ってください。でも「za」よりは「ja」に近います。だから「ja」に書いたんです。でも初めからそこまで言うのは厳しから、最初は簡単に「ざ」の発音を持っているんだと思うことにしましょう。

次は「차」です。英語で「cha」って書いているんですけど、「ちゃ」とはちがう発音です。「ちゃ」は韓国語の「챠」に近い発音です。日本語では書いた通り「つぁ」、これを早く言う方がもっと「차」に近い音です。でも「츠(cheu)」と「つ(tsu)」は少しちがう音です。そのせいで日本語の「つ」は韓国語で「쓰(th)」って書くこともよくあります。例えば「つなみ(쓰나미)」,「ツリー(츠리)」があります。同じ「つ」なのになぜ違うのか気になるか知れませんけど、これはただ「つ」の発音が韓国語に表現できないから曖昧になるだけです。あんまり気にしないほうがいいです。でも「쯔(つ)」は本当に似ている発音です。例えば「つくる」は「쯔꾸르」って言います。またここで「く」を「쿠(くぅ)」ではなく「꾸(く)」って読むことも確認できます。これは全部韓国語と日本語の発音が違うから起こっている現象なんです。

説明が長かったんですけど、韓国語の発音に慣れるとどんな話をしているのかもっと分かり易くなるはずだとおもいます。

次は母音を見てみましょう。

| 番号  | 母音  | 日本語 | 英語  |
| :-: | :-: | :-: | :-: |
| 01  |  ㅏ  |  あ  | ah  |
| 02  |  ㅑ  |  や  | ya  |
| 03  |  ㅓ  |  ぉ  | uh  |
| 04  |  ㅕ  |  ょ  | yeo |
| 05  |  ㅗ  |  お  | oh  |
| 06  |  ㅛ  |  よ  | yo  |
| 07  |  ㅜ  |  う  | ooh |
| 08  |  ㅠ  |  ゆ  | yu  |
| 09  |  ㅡ  |  ぅ  | eu  |
| 10  |  ㅣ  |  い  | yi  |

母音は多分日本人に難しいところです。特に有名なのは「ㅓ/ㅕ」と「ㅗ/ㅛ」の違い、それと「ㅜ」と「ㅡ」の違いです。

まず、「ㅓ」です。もし英語単語の「thunder」を発音できますか？これは韓国語に「썬더」って読むんですけど、見てるとおりに「ㅓ」の発音を2回も確認できます。「サンダー」は韓国語に読むと「산다ー」なので少し違う発音です。

「ㅕ」はどうでしょう。これは「ㅣ(yi)」と「ㅓ(uh)」を早く言った結果と同じです。これは「いぉ」を早く言って「よ」になることで確認できます。だから「ㅓ」だけ言えるようになったら、「ㅕ」も発音できます。

さって、「ㅜ」と「ㅡ」の違いはどうでしょう。「spoon」は日本語に「スプーン」になるんですけど、これは結構似ている発音です。韓国語には「스푼」って書くんですけどこれも結構似ている音です。ここで、「ス」が「스」になり、「プー」が「푸」になることが見えますか？そんな感覚です。

もっと例を言うと、「soup」があります。これは日本語で「スープ」って読みます。韓国語では「수프」です。「スー」が「수」になるのが見えますか？でも「プ」は「푸」ではなく「프」になりました。日本語の「う段」は韓国語の「우」と「으」の間にある発音です。「우」の音はほとんど後ろに「う」をつけるだけで結構似ている音を出すのが可能です。難しいのは「으」なんですけど、これは「す」や「つ」を長く言って「스으」や「쯔으」を発音するんだと思ってみましょう。「すう」、「つう」とは違う音です。それを「す」や「つ」なしで言えたらそれが「으」の発音です。「う(우)」とは違う音ですよね？

ここで少し練習してみましょう。次の表は日本語単語をハングルで書いた例です。今さら言うと韓国語は言語、ハングルは文字です。日本語が言語でガナが文字ですからそれと同じです。だからここは韓国語ではなくハングルで書くって言うのが正確な表現ですね。

| 番号  | ハングル | 日本語  |
| :-: | :--: | :--: |
|  1  | 사쿠라  | さくら  |
|  2  | 나루토  | なると  |
|  3  | 마츠리  | まつり  |
|  4  | 하나비  | はなび  |
|  5  |  라멘  | ラーメン |
|  6  |  우동  | うどん  |

どうですか？読みましたか？日本語をハングルで書くから不思議な感覚はなかったんでしょうか。

でもこれは日本語で言える発音をハングルに書いただけなので、ハングルをそのまま読めるようになるためにはもっと発音の練習が必要です。

今度はハングルにある基本的な子音と母音を話しただけです。バリエーションについてはまだ言ったことがありません。でもその前に学べなっきゃ行けないことがあります。子音は最初だけじゃなく、下にもあります。「ラーメン」が「라멘」になったのがその例のひとつです。これは韓国語で「バッチム」っていうものです。これを学ぶ後こそバリエーションの全てを完成するのができるんです。

韓国語の「バッチム」は本当に日本語と似合わない概念です。英語の「McDonald's」を韓国語では「맥도날드」って読むんですけど、日本語では「マクドナルド」って読むのがその例です。韓国語に書くと「마쿠도나루도」になるんですけど、ここで「バッチム」が本当に一つもないことを確認できます。それほど似合わないっていう話です。

でも「ラーメン」は「라멘」に書くことができました。これは一部分のバッチムは発音できるって言う意味です。

次回は、韓国語に発音できるバッチムは何があって、発音が難しいバッチムは何があるのか、それについて見てみましょう。
