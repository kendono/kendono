
# ハングルで書いてみる日本語

まず、日本語の発音の大部分はハングルで書くことができます。全く同じ発音のわけではないんですけど、結構似てる発音を出せるんです。

例えば、日本語の「あいうえお」は、ハングルで「아이우에오」に書くことができます。同じコツで、「かきくけこ」は「카키쿠케코」、「さしすせそ」は「사시스세소」に書けます。

どころで、ハングルで書いた時、文字の形はどんな風になっているのか見えますか？「あ行」を書く時は「ㅇ(イウン)」がずっとあって、「か行」を書く時は「ㅋ(キウク)」がずっとあるんですよね？そして形が「ㅏㅣㅜㅔㅗ(あいうえお)」で続けているのが見えると思います。

そう、ハングルはまるでパズルみたいにそれぞれのパズルのピスを合わせて文字を作るシステムなんです。結構科学的だとは思いませか？このおかげでハングルは読み方を学ぶことが本当に早いことで有名なんです。

じゃ、ここで全ての五十音をどう書くのか調べてみましょう。

| 五十音 | あ段  | い段  | う段  | え段  | お段  |
| :-: | :-: | :-: | :-: | :-: | :-: |
| あ行  |  아  |  이  |  우  |  에  |  오  |
| か行  |  카  |  키  |  쿠  |  케  |  코  |
| さ行  |  사  |  시  |  스  |  세  |  소  |
| た行  |  타  |  치  |  츠  |  테  |  토  |
| な行  |  나  |  니  |  누  |  네  |  노  |
| は行  |  하  |  히  |  후  |  헤  |  호  |
| ま行  |  마  |  미  |  무  |  메  |  모  |
| や行  |  야  |  -  |  유  |  -  |  요  |
| ら行  |  라  |  리  |  루  |  레  |  로  |
| わ行  |  와  |  -  |  -  |  -  |  오  |

コツが見えますか？しかし二つ知るところがあります。

一つ目は「た行」の「ち」と「つ」が「ㅌ(ティウッ)」の代わりに「ㅊ(チウっ)」を使っているところ。二つ目は「う段」の「す」と「つ」が「ㅜ(ウ)」の代わりに「ㅡ(ゥ)」を使ってるところです。

まず、た行がほかのものと同じように「타티투테토」ってなっているんだと思ってみなしょう。これを読むと「たちつてと」ではなく「たてぃとぅてと」になるます。これは多分アルファベットの「T」からも確認できる現象だと思います。

さって、「ㅜ」と「ㅡ」の違いはなんでしょうか。見た通りに、これは「ウ」と「ゥ」に書くのが最善で、日本語で発音を説明するのは結構難しいことです。ただし、韓国語よりは慣れているはずの英語で説明するのは可能です。

「す」から説明してみましょう。「스」は、「speed」の「s」と似ている音です。あと「수」は、「super」の「su」と似ている音です。日本語で説明すると、「スウ」や「シュ」に似ている音なんだと考えれます。実際に、「スウ」ってよむ漢字「数」は、韓国語で「수」ってよみます。「シュ」は「슈」に近って、またちがう発音です。でも発音するコツが似ているんだと考えれます。

「つ」は二つの変化が同時に起こっている文字です。ほかの文字と同じように書くと「투」や「추」になるはずと思うかも知れません。でも先言った通りに「투」は「トゥ」と似ている発音です。そして「추」は「チュ」と似ている発音です。正確には、「チュ」は「츄」と似ている発音です。「やゆよ」が「야유요」になったことと、「シュ」が「슈」になったことと同じ原理です。

どうですか？説明を聞いたからは、それぞれのがながなぜそうなるのか理解できますか？

日本人には、五十音がハングルにどうなるのかわかるのかいいスタートになれます。これはただハングルがどんな風になっているのか調べてみるのが目的なので、ここに出てる全ての文字を覚える必要があるとか、そんなものではありません。

それより重要なのは、ハングルを日本語に変換する必要なしで、そのまま読むコツを持つことです。これについては、英語を使うのが便利かもしれません。例えば、「あいうえお」は「aiueo」に、「かきくけこ」は「kakikukeko」にかけます。英語も結構、ハングルみたいにパズルみたいな感じがありますよね？だから英語もちょっと使う必要があるんです。

なので次は、英語の力をちょっと借りてハングルにあるパズルのピスをどう読むのか、それについて調べてみます。
