# 韓国語にあるバッチム

基本的に使うバッチムは基本的にある子音と同じです。

「아(あ)」を使って例を見てみましょう。

| 番号  | バッチム |   発音   | 英語  |
| :-: | :--: | :----: | :-: |
| 01  |  악   | \[あぐ]  | ahg |
| 02  |  안   |  (あん)  | ahn |
| 03  |  앋   |   あっ   | aht |
| 04  |  알   | \[ある]  | ahl |
| 05  |  암   |  (あん)  | ahm |
| 06  |  압   | \[あっぶ] | ahb |
| 07  |  앗   |   あっ   | aht |
| 08  |  앙   |  (あん)  | ang |
| 09  |  앚   |   あっ   | aht |
| 10  |  앛   |   あっ   | aht |
| 11  |  앜   | \[あく]  | ahk |
| 12  |  앝   |   あっ   | aht |
| 13  |  앞   | \[あっぷ] | ahp |
| 14  |  앟   |   あっ   | aht |

あら、結構多くのバッチムが「あっ」になっていますね。これは日本語でバッチムの音を正確に表現できないのが理由じゃないです。実際に韓国語でも多くのバッチムが「앗(あっ)」と同じ音を出しています。そして二つのバッチムが「あん」に発音できるんだと書いています。子音のように読み方がいっぱいあるのかと心配したが、いいことですね。

でもここには気をつかなっきゃいけないことがあります。それは「っ」と「ん」が出てる音が単語について違うことです。

「いっぱい」、これを英語に書くとどうなりますか？「ippai」になります。これは韓国語に「입빠이」になります。これは「압(\[あっぶ])」に使ったバッチムと同じバッチムです。

「いっしょに」の方はどうでしょうか。これは「itshoni」に書けます。韓国語では「잇쇼니」です。「앗(あっ)」と同じバッチムです。

両方「っ」を使ったんですけど違うバッチムになれいます。そして英語からでも発音の違いさを知ることができます。

次は「ん」です。「こんにちは」は「konnichiwa」になります。韓国語では「곤니치와」です。そして「こんばんは」は「kombangwa」になります。韓国語では「곰방와」です。

両方「ん」を使っていますが、発音が少し違うんですよね。

一応「あっ」になっているバッチムはそのまま読んでもいいです。\[あぐ]みたいに書いたのは実際には一つの音でよむ、日本語で表現できない難しい発音です。

(あん)みたいになっているのは、そのまま読むことは出来ないけどコツしだいで読める発音です。

- 안 - あん(あんた, ahn-ta)
- 암 - あん(あんぶ, ahm-bu)
- 앙 - あん(あんがい, ang-gai)

さって、残りを見てみましょう。

- 악
- 알
- 압
- 앜
- 앞

「압/앞」から見てみます。これも実は、コツによって「あっ」って読めるバッチムです。先「いっぱい」が「입빠이」になったんですよね。そこで「ぱい」なしで言うと「입」の発音が出ます。「압/앞」の違いは、一文字としてはないんです。後ろにつく文字しだいで変わるだけです。なので「잎」も「입」みたいに言えばいいんです。

「악/앜」も一文字としては同じです。これは少しコツが必要なんですが、それでも言うこと自体はできます。「ちょくせつ(直接)」を言う時、「ちょーくーせーつ」よりは「ちょくーせーつ」みたいに言うんですよね。この時「く」を言うコツを使えばいいんです。同じコツで「あくーせつ」って言うんだと考えてみましょう。そのときの「あく」が「악」の発音なんです。でも「悪」みたいに読むことじゃないから、\[あく]みたいに書いて一つの音で言う方がいいでしょう。実際に漢字「悪」は韓国語で「악」って読みます。「悪魔」は「악마」です。

残りは「알」ですね。もし、英語単語「rule」の発音ができますか？「ルール」じゃなくて「rule」です。これは韓国語では「룰」になります。発音の練習して有名な単語は「水」、「물」があります。

多くな方がこれは日本語で表現できない発音だと知ってることが多いんですが、その「ㄹ(リウル)」バッチムの音を出す瞬間が日本語にもあります。それは歌うときです。

歌うとき、「ら行」の発音を言う前にながく言う時があります。そのとき「ら行」の音は、英語に言うと「r」よりは「l」みたいになります。

「きみといっしょなら」っていう歌詞があるんだと考えてみましょう。ここで「いっしょなら」を普通に長くいうと、「いっーしょーなーら」、英語には「it-sho-na-ra」みたいになるはずです。でも歌うとき、「it-sho-nal-la」みたいに言う時があるんです。ここで「nal」を言ったコツが「ㄹ」バッチムのキーワードなんです。

あと、「물」を言う時「ムル」みたいに言う時が多いんですけど、そこから発音を変えることよりは「ムゥ」から変えるのがもっとましです。

ハングルの面白さは発音する時の舌や口の形を文字に表現するところです。「ㄹ」を言うとき、横から見た舌もそんな風になるんです。「ルール」を発音してみましょう。2回、舌が「ㄹ」の形になるはずです。「ㄹ」バッチムも実はそうです。ただ終わせる方法が少し違うだけです。

「ルール」を言うとき、「ルー」をいいながら「ル」の形だけを作ってみましょう。舌が「ㄹ」から始まって、前に行って、音を止めることなしでまた「ㄹ」の形に戻ったら成功です。できたら英語の「rule」見たな音が出るはずです。それが「룰」の発音です。同じコツで「물」を言ってみましょう。成功したらあなたはもう韓国語で「水」を言える人なんです。

では少しレベルアップした韓国語読みをやってみましょう。今回は日本語じゃない単語もあります。でも似ている単語です。

| 番号  | ハングル |   日本語   |
| :-: | :--: | :-----: |
| 01  | 롭뺘쿠  |  ろっぴゃく  |
| 02  | 쵹세츠  |  ちょくせつ  |
| 03  |  상이  | さんい(三位) |
| 04  | 요오카  |   ようか   |
| 05  |  서부  | せいぶ(西部) |
| 06  |  각지  | かくち(各地) |

どうでしたか？漢字を使うから似ている単語があるところが面白いポイントだと思います。

まだ説明してなかった「ㅃ(サンビウップ)」っていう文字がありますが、これは「いっぱい」を説明するときあったから多分いいんじゃないかと思いました。でもちゃんと学ぶ必要あるでしょう。

今回バッチムを学んだので、次回はハングルにあるバリエーションを覚えてみましょう。これだけ覚えばハングルを全部読めるようになるんです。わくわくしませんか？
